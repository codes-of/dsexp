Steps to run the practical:
If necessary install the Flask module using: pip install flask
The first step is to run the server.py file on a separate terminal.

1. So, open a terminal and type:
    - python server.py (for windows)
    - python3 server.py (for linux)

2. Then open another terminal in the same directory in which both the code file lie. Type:
    - python client.py (for windows)
    - python3 client.py (for linux)
