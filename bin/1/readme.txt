Steps to execute the practical codes:

Open a terminal
1. First compile all the classes using "javac filename" command:
    - javac MyInterface.java
    - javac MyRemoteClass.java
    - javac MyServer.java
    - javac MyClient.java

2. Then run the server code using "java" command:
    - java MyServer

3. Then open a new terminal and then run MyClient:
    - java MyClient

Note: Do not close the previous terminal while using the second terminal.
